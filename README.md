## Desarrollando

1. Clone este repositorio

```bash
git clone git@github.com:vuejs/docs-next.git
```

2. Instale las dependencias

```bash
yarn # o npm install
```

3. Inicie el entorno local de desarrollo

```bash
yarn serve # o npm run serve
```
